Flex-Grow

Permet à un élément (flex-item) de s’élargir pour occuper une partie de l’espace libre restant

La valeur par défaut est 0.

Exemple 1 :

Code

.item01 { flex-grow: 1; }
.item02 { flex-grow: 0; }
.item03 { flex-grow: 0; }
