Order

Par défaut, les items flex sont disposés par ordre d'arrivée. Cependant, la propriété order permet de contrôler l'ordre dans lequel ils apparaissent dans le container.

La propriété justify-content a pour valeur un nombre entier:
order: "un nombre entier"

Example 1 :

Code

.item01 { order: 0; }
.item02 { order: 0; }
.item03 { order: 0; }
.item04 { order: 0; }
.item05 { order: 0; }

N°1 - order : 0
N°2 - order : 0
N°3 - order : 0
N°4 - order : 0
N°5 - order : 0

Example 2 :

Code

.item01 { order: 1; }
.item02 { order: 0; }
.item03 { order: 0; }
.item04 { order: 0; }
.item05 { order: 0; }

N°1 - order : 1
N°2 - order : 0
N°3 - order : 0
N°4 - order : 0
N°5 - order : 0

Example 3 :

Code

.item01 { order: 0; }
.item02 { order: 0; }
.item03 { order: 0; }
.item04 { order: 0; }
.item05 { order: -1; }

N°1 - order : 0
N°2 - order : 0
N°3 - order : 0
N°4 - order : 0
N°5 - order : -1

Example 4 :

Code

.item01 { order: 12; }
.item02 { order: 0; }
.item03 { order: -4; }
.item04 { order: 8; }
.item05 { order: 4; }

N°1 - order : 12
N°2 - order : 0
N°3 - order : -4
N°4 - order : 8
N°5 - order : 4
