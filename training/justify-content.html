justify-content

Elle définit les alignements et centrages sur l’axe principal.

La propriété justify-content a comme valeur :

  • flex-start (valeur par défaut) : Les éléments flexibles groupés au début
  • flex-end : Les éléments flexibles groupés à la fin
  • center : Les éléments groupés au centre
  • space-between : Les Éléments sont répartis de manière égale sur la ligne ; le premier est au début de la ligne, le dernier est à la fin
  • space-around : Les Éléments sont répartis de manière égale sur la ligne avec le même espacement entre chacune

justify-content: flex-start

Code

.flex-container {
display: flex;
justify-content: flex-start;
