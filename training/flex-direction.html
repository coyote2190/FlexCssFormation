Flex-direction

Elle définit le sens d’affichage horizontal ou vertical des éléments flex-item.

La propriété flex-direction a comme valeur :

  • row (par défaut) : distribution horizontale, valeur par défaut
  • row-reverse : distribution horizontale inversée
  • column : distribution verticale
  • column-reverse : distribution verticale inversée

flex-direction: row

Code

.flex-container{
display: flex;
flex-direction: row
}
