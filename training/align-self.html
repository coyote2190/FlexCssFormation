align-self

Elle définit l’alignement d’un élément (flex-item) indépendamment des autres.

La propriété justify-content a comme valeur :

  • flex-start : le premier bord (début) de l'item est placé au début de la ligne transversale
  • flex-end : le dernier bord (fin) de l'item est placé à la fin de la ligne transversale
  • center : l'item est centré sur l'axe transversal
  • baseline : les items sont répartis de manière à ce que leur ligne de base s'aligne
  • stretch (valeur par défaut) : les items s'étirent de façon à remplir le conteneur (tout en respectant min-width/max-width)

align-self: flex-start

Code

.flex-container {
display: flex;
align-items: center;
}
.flex-container .item03{
align-self: flex-start;
}
