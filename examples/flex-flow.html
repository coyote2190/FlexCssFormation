Flex-flow

La propriété CSS flex-flow est une propriété raccourcie pour les propriétés flex-direction et flex-wrap.

Cas 1 : Flex-flow: row wrap

Code

.list{
display: flex;
flex-flow: row wrap;
}

300 g de farine
3 oeufs entiers
3 c. à soupe de sucre
2 c. à soupe d'huile
50 g de beurre fondu
60 cl de lait
5 cl de rhum
