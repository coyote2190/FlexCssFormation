Flex-direction

La propriété flex-direction définit la façon dont les éléments flexibles sont placés dans un conteneur flexible : elle définit l'axe principal et la direction des éléments (normale ou inversée).

La propriété flex-direction a comme valeur :

  • row (par défaut) : distribution horizontale, valeur par défaut
  • row-reverse : distribution horizontale inversée
  • column : distribution verticale
  • column-reverse : distribution verticale inversée

flex-direction: row

Code

.flex-container{
display: flex;
flex-direction: row
}
